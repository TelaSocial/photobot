# photobot
A Telegram bot that receives photo attachments from users and store them
on a different server.

## Requirements

- yarn
  - https://yarnpkg.com/en/docs/install

## Install

```shell
git clone git@github.com:TelaSocial/photobot.git
cd photobot
yarn install
```

## Config
```shell
cp .env-sample .env
$EDITOR .env
```

## Run

```shell
yarn start
```
